# 安装

```
wget https://github.com/prometheus/prometheus/releases/download/v2.1.0/prometheus-2.1.0.darwin-amd64.tar.gz
tar xf prometheus-2.1.0.darwin-amd64.tar.gz
cd prometheus-2.1.0.darwin-amd64
```
